# NTNU_APP_2023_Final
- 師大 112-1 行動 APP 程式設計 期末專題
- 由於不想拉方塊，那就來寫網頁吧

## How to start
1. `git clone git@github.com:Adam7066/NTNU_APP_2023_Final.git`
2. `pnpm install`
3. `pnpm run dev`

## How to build
1. `pnpm run build`
2. upload `/dist/index.html` to app inventor 2
3. set `webviewer` to `http://localhost/index.html`
